# SmartVenue PSU MVP

This is a self-contained front-end MVP for the Beaver Stadium concept described in the provided documents.

## What it includes

- Sign-up first-run flow
- Ticket scan step with Penn State football demo tickets
- Ticket-driven event experience for Beaver Stadium
- Fastest gate recommendation and queue density cards
- Camera-backed crowd intelligence and smart gate signals
- Live zone map, push-style alerts, and operator controls

## How to open it

Open `index.html` in a browser.

## MVP framing

The app is built as a demo for Penn State Football at Beaver Stadium, but the structure is meant to scale to concerts and other high-density venues.
